# Pizza-Hut-Sales-Analysis-SQL-Project

This project analyzes Pizza Hut sales data using SQL to extract meaningful business insights. The objective is to understand revenue trends, customer ordering behavior, and product performance through structured queries.

🎯 Business Objectives

Analyze total revenue and order volume

Identify top-selling pizzas

Evaluate category-wise sales performance

Determine peak order hours

Understand customer purchasing patterns

🗂️ Database Schema

The project uses the following tables:

orders – Order ID, Order Date, Order Time

order_details – Order ID, Pizza ID, Quantity

pizzas – Pizza ID, Pizza Type ID, Size, Price

pizza_types – Pizza Type ID, Name, Category

1️⃣ Basic Analysis

Total number of orders

Total revenue generated

Highest priced pizza

Most common pizza size ordered

2️⃣ Intermediate Analysis

Top 5 best-selling pizzas

Revenue by pizza category

Orders distribution by hour

Daily average orders
